# Lesson template

Scaffold for a new teach-me lesson. At the start of a lesson:

1. Copy this folder to `topics/<topic-slug>/<NNN>-<lesson-slug>/`
   (e.g. `topics/mcp/105-observability/`). Use a bare descriptive slug with no
   number for one-offs (e.g. `topics/mcp/capstone/`).
2. Fill in the titles in `checklist.md` and `raw-notes.md`.
3. Add `deck.html` / `guide.md` and any other artifacts as the lesson produces them.

Files use **bare names** — the folder path already says which topic/lesson it is:
`checklist.md`, `raw-notes.md`, `deck.html`, `guide.md`. See `../../CLAUDE.md`
for the full convention and Definition of Done.
