What the code actually does

Real snippet, plain English

code

function debounce(fn, wait) {
  let timer;
  return (...args) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), wait);
  };
}

plain english

Wrap a function so it waits for a pause in calls before it actually runs.

Every new call cancels whatever run was already scheduled.

...then books a fresh one wait ms in the future.

Rapid-fire calls collapse into a single run — the last one.
